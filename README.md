### Para usar blur en la imagen de i3lock
  - scrot        -- Para la captura de imagenes de i3lock
  - imagemagick  -- Para usar "convert" con scrot "sudo apt-get install imagemagick"
    - Crear la carpeta de  temp en $HOME
    - Dar permiso de ejecucion al script "sudo chmod +x .config/i3/lock/lock_and_blur.sh"
